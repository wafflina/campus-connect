# campus-connect
CampusConnect helps students discover clubs, events, and opportunities tailored to their interests. By personalizing recommendations, it makes it easier for every student to get involved, meet new people, and make the most of campus life.

For design sketch, please see: https://github.com/wafflina/campus-connect/blob/main/campus-connect-design-sketch.pdf

App is deployed here: https://campus-connect-rjqpz7lqjxxmvbqidta2bb.streamlit.app/
